## Do not use as Extension

This repository is **not an extension** in MakeCode. This is **stand alone** program.


## Edit this project

To edit this repository in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/brickhackers/microbit_joystick** and click import
